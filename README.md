NairaNexus 💳

NairaNexus is a fintech web application that allows users to sell gift cards and trade Bitcoin easily while receiving payment directly in Naira.
The platform is designed to provide a fast, secure, and transparent experience for users looking to convert digital assets into local currency.

This project is currently under active development.

🚀 Features

User Authentication (Email & Password)
Secure Dashboard for users
Sell Gift Cards
Bitcoin Trading
Naira Wallet Balance
Add Nigerian Bank Accounts
Withdraw funds to local bank accounts
Transaction History tracking
Admin payment management
Responsive modern UI


🛠 Tech Stack

This project is built using modern frontend technologies:

React.js – Frontend framework
Tailwind CSS – UI styling
Firebase Authentication – User authentication
Firebase Firestore – Database
Firebase Storage – File uploads
React Router – Navigation
Git & GitHub – Version control
